# Кофеин

[![Go Coverage](https://github.com/akovardin/adscoffee/wiki/coverage.svg)](https://raw.githack.com/wiki/akovardin/adscoffee/coverage.html)

Опенсорсная рекламная платформа для веба, игр, мобильных и десктопных приложений. А также для отображения рекламы на страницах документации и в опенсорсных проектах.

Что сейчас реализовано:

- [x] Система плагинов
- [ ] Сервер для подбора рекламы
- [ ] Админка для настройки рекламы
- [ ] Аналитика и отображение статистики
- [ ] Клиент для показа рекламы в приложениях
- [ ] Клиент для показа рекламы на сайте
- [ ] Медиация рекламы

## Простая ротация рекламы

Отдаем баннер по взвешанному рандому на запрос


## Медиация

Получаем наш баннер из ротации и сравниваем его с вариантами из других сеток
